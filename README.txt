このソフトは，時給調査を行うために作成されました．
現在は中国地方にしか対応していません．
今後，47都道府県に対応していきたいと思います．

検索したい県にチェックを入れます．
チェックの入った県の全ての市ごとに時給の調査を開始します．
使用するサイトはタウンワークです．
検索条件は
・給与ー時給
・職種ー飲食/フード
・雇用形態ーアルバイト・パート

検索後さらに
[職種]
ファーストフード
キッチンスタッフ(ファミレス・レストラン)
ファミレス・レストラン(ホールスタッフ)
うどん・蕎麦屋
にチェックを入れて検索．

検索結果を時給順に並べて，上から順にcsv出力を行います．
市ごとにファイルが生成され，ファイルの先頭には検索結果のURLが入る．
その後，最初に指定した店舗数の時給情報等が出力される．
